Additional API Requests

Authentication:

GET Request:

http://domin.com/player_api.php?username=X&password=X

GET Live Stream Categories:

GET Request:

http://domin.com/player_api.php?username=X&password=X&action=get_live_categories

GET VOD Stream Categories:

GET Request:

http://domin.com/player_api.php?username=X&password=X&action=get_vod_categories

GET SERIES Categories:

GET Request:

http://domin.com/player_api.php?username=X&password=X&action=get_series_categories

GET LIVE Streams:

GET Request (All LIVE Streams):

http://domin.com/player_api.php?username=X&password=X&action=get_live_streams

GET Request (All LIVE Streams in selected category ONLY):

http://domin.com/player_api.php?username=X&password=X&action=get_live_streams&category_id=X

GET VOD Streams:

GET Request (All VOD Streams):

http://domin.com/player_api.php?username=X&password=X&action=get_vod_streams

GET Request (All VOD Streams in selected category ONLY):

http://domin.com/player_api.php?username=X&password=X&action=get_vod_streams&category_id=X

GET SERIES Streams:

GET Request (All Series):

http://domin.com/player_api.php?username=X&password=X&action=get_series

GET Request (All Series in selected category ONLY):

http://domin.com/player_api.php?username=X&password=X&action=get_series&category_id=X

GET SERIES Info:

GET Request:

http://domin.com/player_api.php?username=X&password=X&action=get_series_info&series_id=X

GET VOD Info:

GET Request (Info such as video codecs, duration, description, directors for 1 VOD):

http://domin.com/player_api.php?username=X&password=X&action=get_vod_info&vod_id=X

GET short_epg for LIVE Streams:

GET Request:

http://domin.com/player_api.php?username=X&password=X&action=get_short_epg&stream_id=X

GET Request (Specify a limit, default is 4 EPG listings):

http://domin.com/player_api.php?username=X&password=X&action=get_short_epg&stream_id=X&limit=X

GET ALL EPG for LIVE Streams:

GET Request:

http://domin.com/player_api.php?username=X&password=X&action=get_simple_data_table&stream_id=X

Full EPG List for all Streams:

xmltv.php?username=X&password=X

This project requires requests to have the following User Agent:

Roy_Player_App

Make sure to set the User Agent in your requests to ensure proper communication with the project.
